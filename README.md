# Smart USB Relay
Wemos D1 mini/pro smart relay with web client 
## Features
- Reconnect usb 2.0/3.0
- Websever + webclient
- LED status
- Can be used as hostspot or wifi client
- Power over USB port
## Tech
- Wemos d1 mini/pro or esp8266
- C++

## Installation
Sketch can be downloaded over Arduino IDE
